리눅스에서 가상환경을 만든다
1.  DL  디렉토리를 만든다
  >  mkdir DL
  > cd DL
2.  그 안에 들어가서 가상환경 v_DL 을 만든다.
   > python3 -m venv v_DL
3. 가상환경을 실행시킨다
   > source v_DL / bin / activate    

4.  tensorflow 를 설치한다.
   > pip install tensorflow 

5.  jupyterlab 을 설치한다.
  > pip install jupyterlab

6.  jupyter lab 을 실행한다.
  > jupyter lab
