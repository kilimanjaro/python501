Django의 공식적인 설계 패턴은 무엇인가요?  MVT 
Model 이란: 데이터베이스 (= 데이터베이스안의 테이블) 
View 는 : 처리로직 
Templates 는:  화면  

Django 프로젝트 내의 설정을 담당하는 파일은?  settings.py

프로젝트 내의 URL 경로를 매핑하는, 라우팅을 담당하는 routing  파일은? urls.py

django-admin startproject [프로젝트명]  : 프로젝트 만들 때


python manage.py makemigrations
python manage.py migrate  
이 두가지는 각각 무슨 역할?
 
python manage.py startapp [앱이름]      : 앱 만들 때

개발용 서버를 실행하는 명령어는?
python manage.py runserver

python manage.py runserver 8001    

클라이언트의 요청을 받아 응답을 반환하기 위하여 로직을 수행하는 함수나 클래스를 실행하는 과정은?  뷰 View

템플릿에서 변수의 이름이 object일 때, 변수를 표현하는 템플릿 태그는?
${ object }   
{{ object }}

for 문을 템플릿에서 돌리고 싶을 때, 파이썬 코드를 둘러싸는 템플릿 태그는?
   {%  for  object in objects : %}
    .....
   {% endfor  %}
